Here are some of the most common port numbers used by applications and services:

-   21: FTP (File Transfer Protocol)
-   22: SSH (Secure Shell)
-   23: Telnet
-   25: SMTP (Simple Mail Transfer Protocol)
-   53: DNS (Domain Name System)
-   80: HTTP (Hypertext Transfer Protocol)
-   110: POP3 (Post Office Protocol version 3)
-   119: NNTP (Network News Transfer Protocol)
-   123: NTP (Network Time Protocol)
-   143: IMAP (Internet Message Access Protocol)
-   161: SNMP (Simple Network Management Protocol)
-   443: HTTPS (HTTP Secure)
-   465: SMTPS (Secure SMTP)
-   514: Syslog
-   587: Submission (SMTP)
-   993: IMAPS (Secure IMAP)
-   995: POP3S (Secure POP3)

It's important to note that these port numbers are not the only ones used by these applications and services, and that other applications and services may use these port numbers as well. It's also important to keep in mind that some applications may use dynamic port numbers, which are not fixed and can change each time the application is used.
